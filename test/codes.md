# Python Lib

## Numpy

**sum** an array:
```python
import numpy as np  
a=np.array([0.4,0.5])  
b=np.sum(a)  
```

```python
import numpy as np  
a=np.array([[1,4],[3,5]])  
b=np.sum(a,axis=0) 
```
